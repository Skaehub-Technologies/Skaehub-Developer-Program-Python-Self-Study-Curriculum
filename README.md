# SkaeHub Python-Self-Study-Curriculum

This is a guide into the world of sotware development. It is meant for anyone who is interested in learning about software development using Python programming language. You can choose to read specific chapters (for experienced developers) or go through the whole course from beginning to end. This curriculum fully prepares you for the on-site bootcamp.
